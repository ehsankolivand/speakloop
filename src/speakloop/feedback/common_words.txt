# speakloop coherence wordlist (FR-006)
#
# A compact, high-frequency English word list used by feedback/coherence.py to
# decide whether an evidence quote is coherent English or ASR garble. It is the
# project's offline, dependency-free substitute for a language model
# (Constitution Principle II; research.md §e).
#
# Format: one lowercase word per line. Blank lines and lines beginning with "#"
# are ignored by the loader. This is the high-frequency CORE (function words +
# the most common content words and their common inflections, plus the spoken
# software-interview vocabulary that recurs in this product). It is intentionally
# small; the coherence filter additionally treats any term the speaker uses
# repeatedly across attempts as attested (so genuine jargon is never flagged),
# so the list does not need to enumerate all technical terms.
#
# Do NOT add non-words / ASR artefacts here (e.g. "rt"); doing so would defeat
# the garble filter and break tests/unit/feedback/test_coherence.py.
